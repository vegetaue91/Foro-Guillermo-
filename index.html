⚙️ Solo necesitas hacer esto una vez para activar el foro
  1. Ve a console.firebase.google.com → crea un proyecto gratis
  2. En el menú izquierdo abre Firestore Database → "Crear base de datos" → modo prueba
  3. Haz clic en el engranaje ⚙️ → Configuración del proyecto → ícono </> → registra la app
  4. Copia los valores que aparecen y pégalos abajo donde dice TU_*
  5. Sube este archivo a GitHub Pages y comparte el link con tus compañeros
